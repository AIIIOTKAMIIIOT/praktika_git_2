GitHub — это онлайн-сервис для хранения и управления Git-репозиториями.  
Он позволяет разработчикам хранить проекты в интернете и работать над ними совместно.  
Команда git push используется для отправки изменений с локального компьютера на удалённый репозиторий.  
Команда git pull позволяет получить последние изменения с удалённого репозитория.  
GitHub также предоставляет удобный интерфейс для просмотра кода и истории изменений.
